MCP-DevOps Logo Generator

使用说明

  1. 查看 Logo: 页面加载后即可查看 MCP-DevOps 的 Logo。
  2. 下载 PNG 格式: 点击"下载 PNG 格式 Logo"按钮，将以 PNG 格式下载 Logo 图片。
  3. 下载 SVG 格式: 点击"下载 SVG 格式 Logo"按钮，将以 SVG 格式下载 Logo 图片。
  4. Logo 设计说明:
    • 蓝色轮形图案代表 Kubernetes
    • 绿色矩形中的 MCP 代表 Model Context Protocol
    • 蓝色无限符号代表 DevOps 的持续集成与部署
    • 黄色消息图标代表企业微信通知功能
